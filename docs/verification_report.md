## 活用方法
今後開発を進める中で、このテストツールを以下のコマンドで実行することで、意図しない変更（デグレ）を検知できます。
```powershell
cd c:\works\AtkSystem\AtkSystem.Tests
npx playwright test
```
失敗するテストについては、アプリケーションのHTML構造（IDやClass）が変更された際に、テストコード側のセレクタ(`tests/*.spec.js`)を微調整して運用してください。
